The main motto of this website is to allow the learner grasp the core concepts of computer science and mathematical aspects related to computer science, as well as make him/her productive and self-motivated.
This website is for those who are interested/involved in computer science and/or applied mathematics.

I have (from my experience) classified computer science into following three categories.

Theoretical Computer Science
This teaches the core CS stuffs, which involves researchable topics, techniques and algorithms leaving behind the implementation using programming.
Applied Mathematics
This teaches the mathematical stuffs, like graph theory , number theory , stochastic calculus, integrals and linear algebra etc necessary to know for a computer engineer.
Applied Computer Science
This involves the actual programming stuffs, like web application development, implementation of computer graphics using APIs like OpenGL or DirectX , game development using Unity or Unreal etc

Theoretical Computer Science comprises of following topics.

  1. Theory of Computation
  2. Data Structures and Algorithms
  3. Analysis of Algorithms
  4. Designing of a compiler
  5. Digital Electronics and Microprocessors
  6. Computer Organizations and Architecture
  7. Embedded Systems and HDL(Hardware Description Language)
  8. Operating System
  9. Database Management System
  10. Digital Signal Processing; Image processing
  11. Cryptography
  12. Artificial Neural Networks; Machine learning
  13. Classical Computation and Quantum Computation

Applied mathematics involves the following topics.

  1. Linear Algebra
  2. Multivariable Calculus
  3. Multiple Integrals
  4. Integral Transforms
  5. Discrete Mathematics
  6. Probability and Statistics
  7. Stochastic calculus
  8. Numerical Methods for Engineers
  9. Number Theory

Applied Computer Science involves the workflow you used to be involved in , when you land in a job environment.


I have further categorized applied CS into five arenas.

  1. Web Development
  2. Game Development
  3. Application Development
  4. Security Engineering
  5. Data Science
  6. Internet of Things(IoT)
